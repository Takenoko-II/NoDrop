# NoDrop

NoDrop Version 2.0.0

特定のアイテムがドロップされることを封じます

## つかいかた
アイテム修飾子を適用するだけ！！！

```mcfunction
// 例: メインハンド
item modify entity @s weapon.mainhand no_drop:
```

アイテム修飾子 `no_drop:` をアイテムに適用すると、クリエイティブモードでないプレイヤーはそのアイテムに対して以下のことができなくなる:

### コンテナを閉じている状態でのQキーによるドロップ
ドロップを試みると、ドロップする直前のアイテムスロットにアイテムが瞬時に戻ってくる

### コンテナを開いている状態でのQキーによるドロップ
インベントリを閉じているとき同様、ドロップする直前のアイテムスロットにアイテムが瞬時に戻ってくる

### コンテナを開いている状態での持ち上げ→UI外クリックによるドロップ
カーソルスロットに瞬時に戻ってくる

### クラフティングスロット(2×2, 3×3)等の非インベントリコンテナからのQキーによるドロップ
インベントリに `give` コマンドと同様の挙動で戻ってくる
<br>他プレイヤーに拾われる心配はない
<br>インベントリがいっぱいの場合、プレイヤーの目の位置にテレポートさせ続ける仕様のため、そのままログアウトをされるとアイテムだけがそこに残る(再度ログインしてインベントリに空きを作ればいい話ではある)
